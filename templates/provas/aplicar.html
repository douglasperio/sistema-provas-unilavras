Como Funciona

Configure e Gere
O sistema cria versões embaralhadas (A, B, C…), distribui os alunos nas versões e gera um QR Code único para cada um.
2
Imprima o PDF
Baixe o PDF único com todas as provas. Cada folha tem na página 1: QR Code + gabarito de bolhas. As páginas seguintes contêm as questões.
3
Corrija pelo Celular
Abra a tela de correção no celular, escaneie o QR Code da folha do aluno e marque as respostas. A nota é calculada automaticamente.
4
Exporte os Resultados
Baixe um arquivo Excel com todas as notas, gabaritos por versão e situação de cada aluno.
